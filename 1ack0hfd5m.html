Aggregating Citizen Scientist Ratings to Emulate Expert Labels

Due to the varying skill levels of the citizen scientists, we trained a model to aggregate labels based on the rater, in order to create more optimal labels. We used the XGBoost algorithm \cite{Chen2016}, an ensemble method that combines a set of weak learners (decision trees) to predict the "gold" standard labels based on a set of features, which were the average rating of the image from each rater.
